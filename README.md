# Scala-WebRTC-Chat
Proof of concept implementation of P2P chat using WebRTC on Scala.

# Dependencies
- [Scala.js](https://www.scala-js.org/)
- [Scala Cats](https://typelevel.org/cats/)
- [Scala Cats Effect](https://typelevel.org/cats-effect/)
- [FS2](https://fs2.io/)
- [Calico](https://www.armanbilge.com/calico)
